# K-Means-demonstrate

实现K-Means算法的可视化和图片降维

# 作业

## 作业一

做一个demo, 对按一定规律或随机生成的点进行K聚类

## 作业二

输入: 一张图片
使用K-means 算法对颜色进行聚类输出,仅由K种颜色组成的输入图片.



# 使用步骤

## 安装依赖

```
pip install -r requirements.txt

or

pip3 install -r requirements.txt
```

## 作业一

> Kmeans算法实现

按照提醒输入K值即可

## 作业二

> 图片降维

按照提醒输入图片地址和K值即可.默认已经放了一张demo图了.
